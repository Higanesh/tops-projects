6. How to clone repository in local system ? -> Steps to Clone a Repository 1. On GitHub.com, navigate to the main page of the repository. 2. Copy the URL for the repository. 3. Open Git Bash. 4. Change the current working directory to the location where you want the cloned directory. 5. Type git clone, and then paste the URL you copied earlier. eg. git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY 6. Press Enter to create your local clone.
