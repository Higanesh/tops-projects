3. what is high level language ? -> These languages are more abstract and easier for humans to read and write. They are independent of the computer's hardware and are translated into machine code using compilers or interpreters. Procedural Languages: Emphasize a sequence of actions or commands to be performed (e.g., C, Pascal). Object-Oriented Languages: Focus on defining data structures as "objects" with associated procedures known as methods (e.g., Java, C++, Python). Functional Languages: Treat computation as the evaluation of mathematical functions and avoid changing-state and mutable data (e.g., Haskell, Lisp). Scripting Languages: Often used for writing small programs or scripts to automate tasks (e.g., JavaScript, Perl, Python).
